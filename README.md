1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/my-react-app.git
   cd my-react-app
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm start
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to see the app running.
